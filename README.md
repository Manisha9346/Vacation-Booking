# 🌴 Vacation Booking Website

A simple and responsive Vacation Booking web application built using HTML and CSS. This project showcases a travel booking interface where users can explore destinations and view vacation packages.

## ✨ Features
- 🏝️ Attractive homepage with travel destinations
- 📦 Vacation package listings
- 📱 Responsive design for mobile and desktop
- 🎨 Clean UI with modern styling using CSS
- 🧭 Easy navigation structure

## 🛠️ Tech Stack
- HTML5
- CSS3
